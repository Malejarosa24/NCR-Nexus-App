# NCR Nexus App

Aplicación móvil desarrollada en Android Studio para la gestión de solicitudes NCR.

## Tecnologías

- Kotlin
- Android Studio
- Firebase Authentication
- SQLite
- Material Design

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Recuperación de contraseña
- Registro de solicitudes
- Consulta de tickets
- Almacenamiento local mediante SQLite
- Autenticación mediante Firebase
